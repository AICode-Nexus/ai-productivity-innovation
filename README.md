# 企业 AI 提效与创新互动分享站

面向管理者和业务团队的静态互动分享站，覆盖国内外 AI 提效趋势、企业提效地图、泛企业案例、提示词模板和 90 天落地路线图。

## Preview

本项目无构建依赖，直接打开 `index.html` 即可预览；也可以启动本地静态服务：

```bash
python3 -m http.server 4173
```

然后访问 `http://localhost:4173/`。

## Structure

- `index.html`：页面结构与分享内容
- `styles.css`：响应式布局、投屏模式和视觉样式
- `main.js`：自测、场景选择、提示词生成、复制和动态视觉
- `assets/`：站点图标与可选视觉资源
